LOVING HOMES WEBSITE
====================

A complete multi-page website for Loving Homes luxury dog hotel in Hong Kong.

FOLDER STRUCTURE:
-----------------
loving-homes-website/
├── index.html              # Home page
├── facilities.html         # Facilities page  
├── packages.html           # Packages page
├── gallery.html            # Gallery page (create similarly)
├── contact.html            # Contact page (create similarly)
├── help.html               # Help & Advice page (create similarly)
├── book.html               # Booking page (create similarly)
├── css/
│   └── style.css           # Main stylesheet
├── js/
│   └── main.js             # JavaScript
└── images/                 # Add your images here

HOW TO USE:
-----------
1. Create a folder called "loving-homes-website"
2. Create subfolders: "css", "js", "images"
3. Save each HTML file in the main folder
4. Save style.css in the css folder
5. Save main.js in the js folder
6. Add your images to the images folder
7. Open index.html in your browser

TO ADD IMAGES:
--------------
Replace the image placeholders with your own photos:
- hero-dog.jpg - Main hero image
- grooming.jpg - Grooming facility
- rooms.jpg - Luxury rooms
- etc.

CUSTOMIZATION:
--------------
- Edit colors in :root variables in style.css
- Update text content in HTML files
- Add real phone number and email
- Connect contact form to backend service

PAGES INCLUDED:
